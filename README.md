# my neovim settings

## Installation

- follow the installation guide of [packer](https://github.com/wbthomason/packer.nvim) first.

- clone this repo 

``` sh
git clone https://github.com/cyrusn/nvim.git ~/.config/nvim
```

```sh
nvim ~/.config/nvim 
```

- run `:PackerSync` in nvim and restart.
