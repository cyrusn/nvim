# my neovim settings

## Installation

- follow the installation of [packer](https://github.com/wbthomason/packer.nvim) first.

- clone this repo

```sh
git clone https://github.com/cyrusn/nvim.git ~/.config/nvim
```

- run `:PackerSync` in nvim and restart.
